Python Tutorials
There are plently of excellent Python guides availiable online.  We won't attempt a general introduction to Python here but instead list 
some tutorilas, split by whether you are new to programming or an expierenced programmer new to Python.
Python Tutorials for beginners
  • http://docs.python.org/2/tutorial -the offical Python tutorial
  • http://www.learnpython.org -interactive if slightly over-prescriptive tutorial
  • http://learnpythonthehardway.org -unique but detailed guide
  • http://www.codeacademy.com/tracks/python -interactive tutorial-learn to program Python from any web browser
  • http://rexx.com/~dkuhlman/python_101/python_101.html -Brief introduction to Python
Python Tutorials for experienced programmers new to Python:
  • http://docs.python.org/2 -the offical Python documentation
  • http://developers.google.com/edu/python -Google guide to Python-tutorials
  • http://stackoverflow.com/questions/tagged/python -Python questions on an excellent programming questions and answer site
